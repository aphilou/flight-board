Sur la page pricipale ("Flight Board) plusieurs raccourcis clavier sont utilisables :

touche 'a'
Arrête/Lance l'animation Avion
touches haut/bas
Défilement vertical
touches droite/gauche
Rotation de l'avion
touche 'espace'
Lance/Arrête le défilement vertical automatique
touche 'd'
Arrête tout et positionne l'écran dans sont état initial

Dans la zone de menu, à droite, le lien 'Télécharger les données' permet de récupérer les données saisies pour remplacer le fichier 'assets/DbData.json'.

Le bouton 'Hide' permet de cacher le menu. Utiliser la touche F11 pour passer en mode plein écran.
